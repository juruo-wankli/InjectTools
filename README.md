# InjectTools

一款从Ring0和3以及APC注入的提权注入工具。 //开源无🐎,relax!!

## Usage

APC注入 
example:  
InjectTools.exe 你要注入的程序 DLL路径  
InjectTools.exe lsass.exe C:\Users\ASUS\Desktop\artifact_x64.dll
